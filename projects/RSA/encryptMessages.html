Description

RSA algorithm is an asymmetric cryptography algorithm. This means that for RSA we need two keys: public and private. To generate these keys you will need a p and q value, which are both prime. To ensure the security of RSA, it is necessary to use large prime numbers.

Formulas

N = p * q

r = (p - 1) * (q - 1)

E = a prime numbers

D = E -1 mod r   ≡   D * E = 1 mod r

public key = (e,N)

private key = (D,N)
